# classify

## API

- **`fit_naive_bayes`** (Function) — Fit a Naive Bayes model from labeled samples.
- **`tokenize_for_training`** (Function) — Tokenize text into lowercase words for training.
- **`TrainingSample`** (Struct) — Training corpus entry: a single labeled section.
- **`format_double`** (Function) — Format a Double with fixed decimal places.
- **`json_int`** (Function) — Extract an integer from a JSON payload.
- **`json_str`** (Function) — Extract a string from a JSON payload.
- **`ln`** (Function) — Natural logarithm.
