# tokenize

## API

- **`sub_tokenize_vocab`** (Function) — Split text into lowercase words and emit "vocab:" prefixed feature strings.
- **`feature_value`** (Function) — Extract the value part from a feature string.
- **`feature_kind`** (Function) — Extract the kind part from a feature string.
- **`feature_has_value`** (Function) — Check whether a feature string carries a value (contains `:` separator).
