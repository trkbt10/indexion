# embed

## API

- **`TfidfEmbeddingProvider`** (Struct) — TF-IDF based embedding provider.
- **`build_vocabulary`** (Function) — Build vocabulary from a corpus of texts.
- **`embed`** (Function) — Generate a dense embedding vector for the given text.
- **`new`** (Function) — Create a new TfidfEmbeddingProvider with given dimension.
- **`ln`** (Function) — Natural logarithm approximation via series expansion.
- **`cosine_similarity_dense`** (Function) — Compute cosine similarity between two dense vectors.
