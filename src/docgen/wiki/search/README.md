# search

## API

- **`WikiSearchIndex`** (Struct) — Wiki search index — wraps SearchEngine with wiki-specific metadata.
- **`WikiSection`** (Struct) — A searchable section extracted from a wiki page.
- **`save`** (Function) — Serialize the search index to disk.
- **`WikiSearchHit`** (Struct) — A search hit from the wiki index.
