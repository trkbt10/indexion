# types

## API

- **`WikiSourceRef`** (Struct) — Source file reference (e.g., "src/kgf/lexer/lexer.mbt:45-120").
- **`WikiHeading`** (Struct) — A heading extracted from markdown content for ToC.
- **`WikiPage`** (Struct) — A single wiki page with content and metadata.
- **`ManifestPage`** (Struct) — A page entry in wiki.json manifest.
- **`WikiData`** (Struct) — Wiki data ready for serving.
- **`WikiNav`** (Struct) — Navigation tree wrapper.
- **`WikiNavItem`** (Struct) — Navigation tree item.
- **`WikiManifest`** (Struct) — wiki.json manifest.
