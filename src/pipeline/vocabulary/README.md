# vocabulary

## API

- **`measure_divergence`** (Function) — Measure vocabulary divergence between two sets of per-file token arrays.
- **`GapTerm`** (Struct) — A term found in source but absent from target.
- **`DivergenceResult`** (Struct) — Result of vocabulary divergence measurement.
- **`GapConfig`** (Struct) — Configuration for gap term extraction.
- **`default`** (Function)
- **`source_file_tokens`** (Param)
- **`target_file_tokens`** (Param)
- **`config`** (Param)
- **`source_vec`** (Param)
- **`terms`** (Param)
- **`max`** (Param)
- **`apply_limit`** (Function)
- **`s`** (Param)
- **`is_numeric_value`** (Function)
